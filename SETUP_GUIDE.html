Step 2: Add Images (Optional)

For the best experience, add product images to the asset folders. See the Assets Guide section in README.md for details:

  • Logo: Add Amazon logo as main-assets/logo.png
  • Flag: Add Indian flag as main-assets/indian flag.png
  • Products: Add product images to assets/ folder (see README.md for complete list)
Quick Tip: The hero banner already uses an online image URL, so the site will work even without local images. Product cards will show broken images until you add the photos.

Step 3: Free Image Resources

Get free product images from these sites:

  • Unsplash: unsplash.com - High-quality free photos
  • Pexels: pexels.com - Free stock photos
  • Pixabay: pixabay.com - Free images and videos
  • Placeholder: Use https://via.placeholder.com/500x500 for testing

Step 4: Use a Local Server (Recommended)

For better performance, run a local server:

  • VS Code: Install "Live Server" extension and right-click index.html ‚Üí "Open with Live Server"
  • Python: Run python -m http.server 8000 in the terminal
  • Node.js: Run npx http-server in the terminal
